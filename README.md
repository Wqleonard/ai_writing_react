# Boom Cat

React + TypeScript + Vite 项目，集成 Zustand、React Router、Tiptap 编辑器。

## 本地开发

```bash
npm install
npm run dev
```

## 技术栈

- React 19 + TypeScript + Vite
- Zustand（状态管理）
- React Router（路由）
- Tiptap（富文本编辑器）
- Tailwind CSS

## 脚本

- `npm run dev` - 开发服务器
- `npm run build` - 构建
- `npm run preview` - 预览构建结果
- `npm run lint` - ESLint
